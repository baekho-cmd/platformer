# Platformer

웹 버전은 브라우저에서 바로 열어 플레이 가능한 간단한 모멘텀 기반 플랫폼 게임입니다.

---

### 조작

- **PC**
  - `A` / `D`: 이동
  - `W`: 점프
  - `E`: 대시
  - `R`: 리셋

- **모바일**
  - 화면 하단의 터치 버튼으로 `이동 / 점프 / 대시 / 리셋` 사용 가능
